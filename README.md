# 로또 번호 생성 앱

Flutter로 만든 로그인 기능이 있는 로또 번호 생성기. 로그인 후 최대 15개까지 로또 번호 세트를 만들고 관리 

## 기능

### 로그인
- 이메일과 비밀번호로 로그인
- 입력값 검증하고 오류 메시지 표시
- 화면 터치하면 키보드 자동으로 숨김
- 애니메이션 효과

### 로또 번호 생성
- 1부터 45까지 숫자 중에서 중복 없이 6개 랜덤 선택
- 생성된 번호는 자동으로 오름차순 정렬
- 최대 15개까지만 생성 가능 (넘으면 토스트 메시지)
- SVG 이미지로 로또 공 아이콘 표시

### 번호 관리
- 생성한 모든 번호 세트를 리스트로 확인
- 각 항목 클릭하면 팝업으로 상세 보기
- 새로고침 버튼으로 전체 삭제
- 커스텀 스크롤바로 리스트 탐색

### UI
- 모바일, 태블릿, 데스크탑 모두 지원
- Material Design 스타일
- flutter_animate로 애니메이션 효과
- 재사용 가능한 커스텀 위젯들

## 사용 기술

- Flutter 3.9.2 이상
- Dart 3.9.2 이상

주요 패키지:
- flutter_svg: SVG 이미지 표시
- fluttertoast: 토스트 메시지
- flutter_animate: 애니메이션

## 프로젝트 구조

```
lib/
├── main.dart              앱 시작점
├── login.dart             로그인 화면
├── lotto.dart             로또 번호 생성 화면
├── my_container.dart      커스텀 컨테이너
├── my_textfield.dart      커스텀 텍스트 필드
└── widget/
    ├── dialog.dart         팝업 다이얼로그
    ├── lotto_ball.dart    로또 공 위젯
    └── layout_lotto.dart  반응형 레이아웃
```

### 로그인
기본 로그인 정보:
- 이메일: email@email.com
- 비밀번호: 123456

로그인 화면에서 위 정보 입력하면 로또 번호 생성 화면으로 이동

### 번호 생성
우측 하단에 있는 + 버튼 누르면 번호가 생성됩니다. 최대 15개까지 만들 수 있고, 넘으면 토스트 메시지가 뜸. 생성된 번호는 자동으로 정렬

### 번호 확인
리스트에서 각 항목의 우측 아이콘을 누르면 팝업으로 상세 정보를 볼 수 있음

### 초기화
새로고침 버튼(빨간색)을 누르면 생성한 모든 번호가 삭제

## 커스터마이징

로그인 정보 바꾸려면 lib/login.dart 파일의 _login() 메서드에서 수정

번호 생성 개수 제한 바꾸려면 lib/lotto.dart의 _createNumber() 메서드에서 숫자 변경
